# Smart-Link-Highlighter-Enhancer
# 网页文本链接智能高亮与增强
[![License](https://img.shields.io/badge/License-MPL_2.0-blue.svg)](LICENSE)
### 📖 简介

这是一个功能强大的油猴脚本，旨在提升网页浏览体验。它能智能识别网页中未被处理的各种纯文本链接（如 `http://`、`www.` 开头的文本），将其转换为可点击的超链接，并提供极致细化的样式定制、网页标题获取、黑白名单过滤等功能。
告别网页中那些“看得见却点不了”的纯文本链接，让浏览更高效。

---

### ✨ 核心功能
#### 1. 智能识别与转换
*   **文本扫描**：自动遍历网页 DOM，识别符合规则的纯文本 URL。
*   **动态监听**：支持 AJAX 动态加载的内容，即使是后续加载的评论区、动态列表也能即时识别。
*   **冲突处理**：
    *   **让步模式**：检测到已有 `<a>` 标签包裹时，保留原样，避免破坏网页结构。
    *   **覆盖模式**：强制移除原有链接属性，统一应用脚本样式。
#### 2. 精细化样式配置 (v2.0+ 重构)
分为 **“默认状态”** 和 **“光标悬停时”** 两套独立配置，每套均支持：
*   **文本颜色**：开关 + 色值选择器。
*   **边框样式**：开关 + 边框颜色选择器。
*   **下划线**：开关控制（颜色自动跟随文本颜色）。
*   **字体样式**：斜体、加粗开关。
#### 3. 网页标题获取 (高级功能)
*   **自动替换**：开启后，脚本会异步请求链接指向的网页，提取 `<title>` 内容并替换原链接文本。
*   **正则过滤**：支持标准 JS 正则表达式，用于清洗标题（如移除“ - 网站名”后缀）。
*   **快捷预设**：内置“移除后缀”、“移除括号”、“移除豆瓣后缀”等一键填入按钮。
*   **智能缓存**：避免重复请求同一 URL，提升性能。
#### 4. 网站黑白名单
*   **黑名单模式**：默认全局启用，名单中的网站禁用脚本。
*   **白名单模式**：默认全局禁用，仅在名单中的网站启用。
*   **快捷操作**：一键“加入当前网站”按钮，自动提取域名。

### 📸 界面预览
配置面板采用侧边悬浮设计，布局清晰，支持滚动查看。
*   **样式配置区**：左右/上下分栏，直观对比默认与悬停效果。

### 🛠 安装方法
1.  安装用户脚本管理器（如 [Tampermonkey](https://www.tampermonkey.net/) 或 Violentmonkey）。
2.  点击页面上方的“安装此脚本”或 [点击此处直接安装](#)。
3.  刷新目标网页，脚本即可自动生效。

### 📝 使用指南
#### 如何打开设置？
点击浏览器右上角的油猴图标，在菜单中选择 **“⚙️ 链接增强设置”**。
#### 正则表达式说明
脚本支持标准的 JavaScript 正则语法：
*   **格式**：`/pattern/flags` (推荐) 或 普通字符串。
*   **示例**：
    *   `/\s*-\s*.*$/` ：匹配横线及之后的所有内容（常用于移除“ - 网站名”）。
    *   `/[(\s\[【].*?[)\s]】]/g` ：匹配中文/英文括号及其内容。
#### 下划线颜色逻辑
为了保持视觉一致性，**下划线颜色强制跟随文本颜色**。如果你修改了文本颜色，下划线会自动变为相同颜色，无需单独设置。

---

### ⚠️ 注意事项
1.  **性能影响**：“获取网页标题”功能会发起网络请求。如果一个页面包含大量链接，可能会增加网络负载。建议仅在需要时开启，或配合黑名单使用。
2.  **跨域限制**：部分网站有严格的跨域策略，可能导致标题获取失败。
3.  **样式优先级**：脚本使用 `!important` 确保样式生效，但在部分结构极其复杂的网站上可能会被覆盖。
