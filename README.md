# Smart-Link-Highlighter-Enhancer 
# 网页文本链接智能高亮与增强
[![License](https://img.shields.io/badge/License-MPL_2.0-blue.svg)](LICENSE)
### 📖 简介

这是一个功能强大的油猴脚本，旨在提升网页浏览体验。它能智能识别网页中未被处理的各种纯文本链接（如 `http://`、`www.` 开头的文本，甚至 `baidu.com` 等裸域名），将其转换为可点击的超链接，并提供极致细化的样式定制、网页标题获取、黑白名单过滤等功能。
**v3.7 更新亮点**：新增“开发者选项”面板，开放了从正则核心参数到 DOM 遍历规则的数十项底层配置，支持裸域名智能识别、防抖优化、显示截断等高级功能。
告别网页中那些“看得见却点不了”的纯文本链接，让浏览更高效。

---
### ✨ 核心功能
#### 1. 智能识别与转换
*   **全面兼容**：支持 `http/https`、`www.` 前缀，以及 **无前缀的裸域名**（如 `baidu.com`）。
*   **高级识别**：支持 IP 地址（如 `192.168.1.1`）、`localhost` 及 Unicode 国际化域名（如中文域名）。
*   **智能边界**：精确区分域名后缀与普通单词（如能正确识别 `minecraft.network` 中的 `network` 不是域名后缀），避免误判。
*   **动态监听**：支持 AJAX 动态加载的内容，即使是后续加载的评论区、动态列表也能即时识别。
*   **防抖机制**：针对频繁更新的网页（如实时聊天室），内置防抖功能避免性能损耗。
*   **冲突处理**：
    *   **让步模式**：检测到已有 `<a>` 标签包裹时，保留原样。
    *   **覆盖模式**：强制移除原有链接属性，统一应用脚本样式。
#### 2. 精细化样式配置
分为 **“默认状态”** 和 **“光标悬停时”** 两套独立配置，每套均支持：
*   **文本颜色**：开关 + 色值选择器。
*   **边框样式**：开关 + 边框颜色、圆角、厚度、偏移量（使用 outline 实现，不破坏布局）。
*   **下划线**：开关控制（颜色自动跟随文本颜色）。
*   **字体样式**：斜体、加粗开关。
#### 3. 网页标题获取 (高级功能)
*   **自动替换**：开启后，脚本会异步请求链接指向的网页，提取 `<title>` 内容并替换原链接文本。
*   **自定义提取**：支持在开发者选项中自定义提取正则，适配特殊网页结构。
*   **正则过滤**：支持标准 JS 正则表达式，用于清洗标题（如移除“ - 网站名”后缀）。
*   **快捷预设**：内置“移除后缀”、“移除括号”、“移除豆瓣后缀”等一键填入按钮。
*   **智能缓存**：支持自定义缓存上限与清理策略，避免内存泄漏。
#### 4. 网站黑白名单
*   **黑名单模式**：默认全局启用，名单中的网站禁用脚本。
*   **白名单模式**：默认全局禁用，仅在名单中的网站启用。
*   **快捷操作**：一键“加入当前网站”按钮，自动提取域名。
#### 5. ⚠️ 开发者选项 (v3.7+ 新增)
为高级用户提供了极其详尽的底层控制权，包含但不限于：
*   **正则核心构造**：自定义匹配协议（支持 ftp/ed2k 等）、匹配前缀、域名主体字符集、TLD 后缀列表。
*   **显示控制**：设置最大显示长度（自动截断超长链接）、Wrapper 的 Margin/Padding、过渡动画时间。
*   **DOM 遍历**：自定义排除的 HTML 标签、排除的 CSS 类名（精确避开特定区域）。
*   **性能调优**：调整防抖延迟、标题获取超时时间、缓存清理比例。

### 📝 使用指南
#### 如何打开设置？
点击浏览器右上角的油猴图标，在菜单中选择 **“⚙️ 链接增强设置”**。
#### 正则表达式说明
脚本支持标准的 JavaScript 正则语法：
*   **格式**：`/pattern/flags` (推荐) 或 普通字符串。
*   **示例**：
    *   `/\s*-\s*.*$/` ：匹配横线及之后的所有内容（常用于移除“ - 网站名”）。
    *   `/[(\s\[【].*?[)\s]】]/g` ：匹配中文/英文括号及其内容。
#### 开发者选项说明
在配置面板底部，点击展开“开发人员选项”。请注意，修改这些选项需要您对正则表达式和 DOM 结构有一定的了解：
*   **TLD 边界检查**：用于防止 `minecraft.network` 被误识别为链接，默认检查字母数字，可根据需要调整。
*   **排除类名**：如果您发现某些特定区域的链接处理有问题，可以通过填写 CSS 类名来排除该区域。
---
### ⚠️ 注意事项
1.  **性能影响**：“获取网页标题”功能会发起网络请求。如果一个页面包含大量链接，可能会增加网络负载。建议配合“最大显示长度”和“防抖延迟”优化体验。
2.  **跨域限制**：部分网站有严格的跨域策略，可能导致标题获取失败。
3.  **开发者选项风险**：错误的正则表达式可能导致脚本失效或网页卡顿，修改前请备份配置（重置脚本配置可恢复默认）。
4.  **样式优先级**：脚本使用 `!important` 确保样式生效，但在部分结构极其复杂的网站上可能会被覆盖。
